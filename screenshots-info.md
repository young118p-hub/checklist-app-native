## 📱 주요 화면들

### 1. 홈 스크린
- 빨간색 헤더 (#DC2626)
- "아맞다이거! 🤦‍♂️" 타이틀
- "깜빡할 뻔한 모든 것들을 한 번에!" 서브타이틀
- 체크리스트 카드들 (흰색 배경, 그림자)

### 2. 체크리스트 상세 화면
- 헤더: 체크리스트 제목
- 체크박스가 있는 아이템 리스트
- 각 아이템: 제목, 설명, 수량
- 완료/미완료 상태별 시각적 구분

### 3. 체크리스트 생성 화면
- 제목 입력 필드
- 설명 입력 필드
- 템플릿 선택 옵션
- 아이템 추가 버튼

### 4. 공유 모달
- 체크리스트 제목 표시
- 공유 방법 선택 (링크 복사, 메신저 공유)
- QR 코드 옵션
